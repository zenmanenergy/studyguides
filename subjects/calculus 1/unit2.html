Calculus 1 — Unit 2: Differentiation: Definition and Fundamental Properties

Definition of the Derivative

\[ f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]
  • Geometrically: slope of the tangent line at \(x\)
  • Physically: instantaneous rate of change
  • Notations: \(f'(x),\ \dfrac{dy}{dx},\ \dfrac{d}{dx}[f(x)],\ \dot{y}\)
  • A function is differentiable at \(x\) only if it is continuous there (converse is NOT true)
  • Not differentiable at: corners, cusps, vertical tangents, discontinuities

Basic Differentiation Rules

RuleFormula
Constant\(\dfrac{d}{dx}[c] = 0\)
Power\(\dfrac{d}{dx}[x^n] = nx^{n-1}\)
Constant Multiple\(\dfrac{d}{dx}[cf] = cf'\)
Sum / Difference\(\dfrac{d}{dx}[f \pm g] = f' \pm g'\)
Product\(\dfrac{d}{dx}[fg] = f'g + fg'\)
Quotient\(\dfrac{d}{dx}\!\left[\dfrac{f}{g}\right] = \dfrac{f'g - fg'}{g^2}\)

Higher-Order Derivatives

  • \(f''(x)\) = second derivative = rate of change of \(f'\) = concavity indicator
  • Notation: \(f^{(n)}(x),\ \dfrac{d^n y}{dx^n}\)
  • Position \(s(t)\) → velocity \(v(t) = s'(t)\) → acceleration \(a(t) = v'(t) = s''(t)\)
