Calculus 2 — Unit 2: Techniques of Integration

7.1 Integration by Parts

Integration by parts comes from the product rule for derivatives. It converts one integral into another that is (hopefully) easier.

\[ \int u\, dv = uv - \int v\, du \]

LIATE rule for choosing \(u\) (pick the first type that appears):

LIATE
LogarithmsInverse trigAlgebraic (polynomials)TrigonometricExponential
  • Whatever you choose as \(u\), differentiate to get \(du\); whatever remains is \(dv\), integrate to get \(v\)
  • Tabular method: when a polynomial multiplies a sine, cosine, or exponential, build a table of repeated derivatives of \(u\) and repeated integrals of \(dv\), then alternate signs
  • Sometimes IBP must be applied twice (e.g., \(\int x^2 e^x\, dx\))
  • Cycling trick: apply IBP twice; if the original integral reappears, move it to one side and solve (e.g., \(\int e^x \sin x\, dx\))
\[ \int e^x \sin x\, dx = \frac{e^x(\sin x - \cos x)}{2} + C \]

7.2 The Partial Fractions Method

Decomposes a rational function \(P(x)/Q(x)\) (with \(\deg P < \deg Q\)) into simpler fractions that integrate easily. If \(\deg P \ge \deg Q\), do polynomial long division first.

Factor in \(Q(x)\)Partial Fraction Term(s)
Linear \((ax+b)\)\(\dfrac{A}{ax+b}\)
Repeated linear \((ax+b)^n\)\(\dfrac{A_1}{ax+b} + \dfrac{A_2}{(ax+b)^2} + \cdots + \dfrac{A_n}{(ax+b)^n}\)
Irreducible quadratic \((ax^2+bx+c)\)\(\dfrac{Ax+B}{ax^2+bx+c}\)
Repeated irreducible quadratic \((ax^2+bx+c)^n\)\(\dfrac{A_1 x+B_1}{ax^2+bx+c} + \cdots + \dfrac{A_n x+B_n}{(ax^2+bx+c)^n}\)
  • Find unknown constants by multiplying both sides by \(Q(x)\), then either equating coefficients or substituting convenient values of \(x\)
  • Integrate each piece: \(\int \frac{A}{ax+b}\,dx = \frac{A}{a}\ln|ax+b| + C\)
  • Irreducible quadratic pieces use completing the square and \(\arctan\)

7.3 Trigonometric Integrals

Integrals of the form \(\displaystyle\int \sin^m x \cos^n x\, dx\):

CaseStrategy
\(m\) oddFactor one \(\sin x\), substitute \(u = \cos x\), use \(\sin^2 x = 1 - \cos^2 x\)
\(n\) oddFactor one \(\cos x\), substitute \(u = \sin x\), use \(\cos^2 x = 1 - \sin^2 x\)
Both evenUse half-angle identities: \(\sin^2 x = \frac{1-\cos 2x}{2}\), \(\cos^2 x = \frac{1+\cos 2x}{2}\)

Integrals of the form \(\displaystyle\int \tan^m x \sec^n x\, dx\):

CaseStrategy
\(n\) even (\(n \ge 2\))Factor \(\sec^2 x\), substitute \(u = \tan x\), use \(\sec^2 x = 1 + \tan^2 x\)
\(m\) oddFactor \(\sec x \tan x\), substitute \(u = \sec x\), use \(\tan^2 x = \sec^2 x - 1\)

Key identities to memorize:

\[ \sin^2 x + \cos^2 x = 1 \qquad \tan^2 x + 1 = \sec^2 x \qquad 1 + \cot^2 x = \csc^2 x \] \[ \sin^2 x = \frac{1 - \cos 2x}{2} \qquad \cos^2 x = \frac{1 + \cos 2x}{2} \qquad \sin x \cos x = \frac{\sin 2x}{2} \]

7.4 Trigonometric Substitutions

Used when the integrand contains \(\sqrt{a^2 - x^2}\), \(\sqrt{a^2 + x^2}\), or \(\sqrt{x^2 - a^2}\).

ExpressionSubstitutionIdentity UsedRange of \(\theta\)
\(\sqrt{a^2 - x^2}\)\(x = a\sin\theta\)\(1 - \sin^2\theta = \cos^2\theta\)\(-\pi/2 \le \theta \le \pi/2\)
\(\sqrt{a^2 + x^2}\)\(x = a\tan\theta\)\(1 + \tan^2\theta = \sec^2\theta\)\(-\pi/2 < \theta < \pi/2\)
\(\sqrt{x^2 - a^2}\)\(x = a\sec\theta\)\(\sec^2\theta - 1 = \tan^2\theta\)\(0 \le \theta < \pi/2\) or \(\pi/2 < \theta \le \pi\)
  • After integrating in \(\theta\), convert back to \(x\) using a right triangle drawn from the substitution
  • Completing the square can turn a general quadratic under a radical into one of the three standard forms

7.5 Integration Strategy

When you see an integral, try techniques in this order:

  1. Simplify first: algebra, trig identities, expand the product
  2. Basic formulas: does it match a standard integral directly?
  3. Substitution (\(u\)-sub): look for a function and its derivative
  4. Integration by parts: product of two different types
  5. Partial fractions: rational function
  6. Trig integrals / trig substitution: powers of trig functions or radical expressions
  7. Manipulate further: multiply by conjugate, use an identity, or split the fraction
  • CAS (Computer Algebra Systems) like Wolfram Alpha can compute integrals but may give forms different from what you expect -- always verify by differentiating the result

7.6 Numerical Integration

When an antiderivative cannot be found in closed form, approximate the definite integral numerically. Divide \([a,b]\) into \(n\) subintervals of width \(\Delta x = (b-a)/n\).

\[ \textbf{Midpoint Rule:} \quad M_n = \Delta x \sum_{i=1}^{n} f\!\left(\bar{x}_i\right) \qquad \bar{x}_i = \text{midpoint of } [x_{i-1}, x_i] \]
\[ \textbf{Trapezoidal Rule:} \quad T_n = \frac{\Delta x}{2}\left[f(x_0) + 2f(x_1) + 2f(x_2) + \cdots + 2f(x_{n-1}) + f(x_n)\right] \]
\[ \textbf{Simpson's Rule:} \quad S_n = \frac{\Delta x}{3}\left[f(x_0) + 4f(x_1) + 2f(x_2) + 4f(x_3) + \cdots + 4f(x_{n-1}) + f(x_n)\right] \quad (n \text{ even}) \]

Error bounds (where \(|f''(x)| \le K\) and \(|f^{(4)}(x)| \le M\) on \([a,b]\)):

\[ |E_T| \le \frac{K(b-a)^3}{12n^2}, \qquad |E_M| \le \frac{K(b-a)^3}{24n^2}, \qquad |E_S| \le \frac{M(b-a)^5}{180n^4} \]
  • Simpson's Rule is exact for polynomials of degree \(\le 3\)
  • Doubling \(n\) cuts the Trapezoidal error by a factor of 4; cuts Simpson's error by a factor of 16

7.7 Improper Integrals

An integral is improper if the interval of integration is infinite or the integrand has a vertical asymptote on \([a,b]\).

Type 1 (infinite limits):

\[ \int_a^{\infty} f(x)\, dx = \lim_{t \to \infty} \int_a^t f(x)\, dx \]

Type 2 (discontinuous integrand): If \(f\) has an asymptote at \(x = b\):

\[ \int_a^b f(x)\, dx = \lim_{t \to b^-} \int_a^t f(x)\, dx \]

The integral converges if the limit exists and is finite; otherwise it diverges.

p-integral test:

\[ \int_1^{\infty} \frac{1}{x^p}\, dx \begin{cases} \text{converges} & p > 1 \\ \text{diverges} & p \le 1 \end{cases} \qquad \int_0^1 \frac{1}{x^p}\, dx \begin{cases} \text{converges} & p < 1 \\ \text{diverges} & p \ge 1 \end{cases} \]

Comparison Theorem: If \(0 \le f(x) \le g(x)\) for \(x \ge a\):

  • If \(\int_a^\infty g(x)\,dx\) converges, so does \(\int_a^\infty f(x)\,dx\)
  • If \(\int_a^\infty f(x)\,dx\) diverges, so does \(\int_a^\infty g(x)\,dx\)
