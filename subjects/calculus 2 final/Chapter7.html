Topic 1 — Improper Integrals and Convergence

Evaluate improper integrals with infinite limits or discontinuous integrands. Use limits: \(\int_a^\infty f(x)\, dx = \lim_{t \to \infty} \int_a^t f(x)\, dx\).

Question 1

Determine whether the following integral is convergent or divergent. If the integral is convergent, find its value. Write the exact answer. Do not round. If it is divergent, write DIV for your answer. \[ \int_{0}^{16} \frac{4}{\sqrt{x}}\, dx \]

Walkthrough

Step 1 — Identify the improper integral.

The integrand \(\frac{4}{\sqrt{x}} = 4x^{-1/2}\) has a singularity at \(x = 0\) (in the interval \([0,16]\)).

Step 2 — Use the limit definition.

\[ \int_0^{16} \frac{4}{\sqrt{x}}\, dx = \lim_{a \to 0^+} \int_a^{16} 4x^{-1/2}\, dx \]

Step 3 — Evaluate the antiderivative.

\(\int 4x^{-1/2}\, dx = 4 \cdot 2x^{1/2} = 8\sqrt{x}\)

Step 4 — Apply limits.

\(\lim_{a \to 0^+} \left[ 8\sqrt{x} \right]_a^{16} = \lim_{a \to 0^+} (8\sqrt{16} - 8\sqrt{a}) = 32 - 0 = 32\)

The integral is convergent: \(\int_0^{16} \frac{4}{\sqrt{x}}\, dx = 32\)

Topic 2 — Integration by Parts

Use the formula \(\int u\, dv = uv - \int v\, du\) to integrate products. Choose \(u\) using LIATE: Log, Inverse trig, Algebraic, Trig, Exponential.

Question 2

Evaluate the integral \[ \int 2t^3 e^{2t}\, dt \] Use \(C\) for the constant of integration. Write the exact answer. Do not round. If necessary, use integration by parts more than once.

Walkthrough

Step 1 — First integration by parts.

Let \(u = 2t^3\), \(dv = e^{2t}\, dt\). Then \(du = 6t^2\, dt\), \(v = \frac{e^{2t}}{2}\).

\[ \int 2t^3 e^{2t}\, dt = t^3 e^{2t} - \int 3t^2 e^{2t}\, dt \]

Step 2 — Second integration by parts.

Let \(u = 3t^2\), \(dv = e^{2t}\, dt\). Then \(du = 6t\, dt\), \(v = \frac{e^{2t}}{2}\).

\(\int 3t^2 e^{2t}\, dt = \frac{3t^2 e^{2t}}{2} - \int 3t e^{2t}\, dt\)

Step 3 — Third integration by parts.

Let \(u = 3t\), \(dv = e^{2t}\, dt\). Then \(du = 3\, dt\), \(v = \frac{e^{2t}}{2}\).

\(\int 3t e^{2t}\, dt = \frac{3te^{2t}}{2} - \int \frac{3e^{2t}}{2}\, dt = \frac{3te^{2t}}{2} - \frac{3e^{2t}}{4}\)

Step 4 — Combine all parts.

\[ \int 2t^3 e^{2t}\, dt = t^3 e^{2t} - \frac{3t^2 e^{2t}}{2} + 3t e^{2t} - \frac{3e^{2t}}{4} + C = e^{2t}\!\left(t^3 - \frac{3t^2}{2} + 3t - \frac{3}{4}\right) + C \]

\(\int 2t^3 e^{2t}\, dt = e^{2t}\!\left(t^3 - \dfrac{3t^2}{2} + 3t - \dfrac{3}{4}\right) + C\)

Topic 3 — Partial Fraction Decomposition

Decompose rational functions into simpler fractions. For linear factors: \(\dfrac{A}{x-a} + \dfrac{B}{x-b}\). Use the Heaviside cover-up method to find coefficients.

Question 3

Apply the Heaviside cover-up method to evaluate the integral \[ \int \frac{4(x + 3)}{(x - 3)(x^2 - 1)}\, dx \] Use \(C\) for the constant of integration. Write the exact answer. Do not round.

Walkthrough

Step 1 — Factor the denominator.

\(x^2 - 1 = (x-1)(x+1)\), so the denominator is \((x-3)(x-1)(x+1)\).

Step 2 — Set up partial fractions.

\[ \frac{4(x+3)}{(x-3)(x-1)(x+1)} = \frac{A}{x-3} + \frac{B}{x-1} + \frac{C}{x+1} \]

Step 3 — Use cover-up method to find \(A\), \(B\), \(C\).

Cover \((x-3)\): \(A = \frac{4(3+3)}{(3-1)(3+1)} = \frac{24}{8} = 3\)

Cover \((x-1)\): \(B = \frac{4(1+3)}{(1-3)(1+1)} = \frac{16}{-4} = -4\)

Cover \((x+1)\): \(C = \frac{4(-1+3)}{(-1-3)(-1-1)} = \frac{8}{8} = 1\)

Step 4 — Integrate.

\[ \int \left( \frac{3}{x-3} - \frac{4}{x-1} + \frac{1}{x+1} \right) dx = 3\ln|x-3| - 4\ln|x-1| + \ln|x+1| + C \]

\(\int \frac{4(x+3)}{(x-3)(x^2-1)}\, dx = 3\ln|x-3| - 4\ln|x-1| + \ln|x+1| + C\)

Topic 4 — Trigonometric Integrals

Integrate powers of sine and cosine. For odd powers, factor out one and use substitution. For even powers, use power reduction formulas.

Question 4

Evaluate the indefinite integral \[ \int 2\sqrt[3]{\cos x}\, \sin^3 x\, dx \] Use \(C\) for the constant of integration. Write the exact answer. Do not round.

Walkthrough

Step 1 — Rewrite using exponent notation.

\(\int 2(\cos x)^{1/3} \sin^3 x\, dx = \int 2(\cos x)^{1/3} \sin^2 x \sin x\, dx\)

Step 2 — Use \(\sin^2 x = 1 - \cos^2 x\).

\(\int 2(\cos x)^{1/3} (1 - \cos^2 x) \sin x\, dx\)

Step 3 — Substitute \(u = \cos x\), \(du = -\sin x\, dx\).

\[ \int 2u^{1/3} (1 - u^2)(-du) = -2\int (u^{1/3} - u^{7/3})\, du \]

Step 4 — Integrate.

\(-2 \left[ \frac{3u^{4/3}}{4} - \frac{3u^{10/3}}{10} \right] = -\frac{3u^{4/3}}{2} + \frac{3u^{10/3}}{5}\)

Step 5 — Back-substitute.

\(-\frac{3(\cos x)^{4/3}}{2} + \frac{3(\cos x)^{10/3}}{5} + C\)

\(\int 2\sqrt[3]{\cos x}\, \sin^3 x\, dx = -\dfrac{3(\cos x)^{4/3}}{2} + \dfrac{3(\cos x)^{10/3}}{5} + C\)

Topic 5 — Integration by Parts (Linear times Trig)

For \(\int p(x)\sin x\, dx\) or \(\int p(x)\cos x\, dx\) where \(p(x)\) is a polynomial, apply integration by parts with \(u = p(x)\), \(dv = \sin x\, dx\) (or \(\cos x\, dx\)).

Question 5

Evaluate the integral \[ \int 3x \sin x\, dx \] Use \(C\) for the constant of integration. Write the exact answer. Do not round.

Walkthrough

Step 1 — Apply integration by parts.

Let \(u = 3x\), \(dv = \sin x\, dx\). Then \(du = 3\, dx\), \(v = -\cos x\).

Step 2 — Use the formula.

\[ \int 3x \sin x\, dx = -3x \cos x - \int -3\cos x\, dx = -3x \cos x + 3\int \cos x\, dx \]

Step 3 — Integrate \(\cos x\).

\(3\int \cos x\, dx = 3\sin x\)

Step 4 — Combine.

\(\int 3x \sin x\, dx = -3x \cos x + 3\sin x + C\)

\(\int 3x \sin x\, dx = -3x \cos x + 3\sin x + C\)

Topic 6 — Improper Integrals at Infinity

Evaluate integrals with infinite upper or lower limits. Test for convergence using power rule: \(\int_a^\infty \frac{1}{x^p}\, dx\) converges if \(p > 1\).

Question 6

Determine whether the improper integral is convergent or divergent. If the improper integral is convergent, evaluate. \[ \int_{3}^{\infty} \frac{4}{x^3}\, dx \]

Walkthrough

Step 1 — Set up with limit.

\(\int_3^\infty \frac{4}{x^3}\, dx = \lim_{t \to \infty} \int_3^t 4x^{-3}\, dx\)

Step 2 — Integrate.

\(\int 4x^{-3}\, dx = 4 \cdot \frac{x^{-2}}{-2} = -\frac{2}{x^2}\)

Step 3 — Apply limits.

\[ \lim_{t \to \infty} \left[ -\frac{2}{x^2} \right]_3^t = \lim_{t \to \infty} \left( -\frac{2}{t^2} + \frac{2}{9} \right) = 0 + \frac{2}{9} = \frac{2}{9} \]

The integral converges to \(\dfrac{2}{9}\)

Topic 7 — Trigonometric Identities and Simplification

Use identities like \(\tan^2 x + 1 = \sec^2 x\) and \(\dfrac{\sec^2\theta}{\csc^2\theta} = \tan^2\theta\) to simplify integrands before integration.

Question 7

Evaluate \[ \int \frac{4\bigl[\tan^2(9x) + 1\bigr]}{\csc^2(9x)}\, dx \] Whenever possible, try to simplify before integrating. Use \(C\) for the constant of integration. Write the exact answer. Do not round.

Walkthrough

Step 1 — Simplify using \(\tan^2\theta + 1 = \sec^2\theta\).

\(\tan^2(9x) + 1 = \sec^2(9x)\)

Step 2 — Substitute.

\[ \int \frac{4\sec^2(9x)}{\csc^2(9x)}\, dx = \int 4\tan^2(9x)\, dx \]

Step 3 — Use \(\tan^2\theta = \sec^2\theta - 1\).

\[ \int 4[\sec^2(9x) - 1]\, dx = 4\int \sec^2(9x)\, dx - 4\int dx \]

Step 4 — Integrate.

\(4 \cdot \frac{\tan(9x)}{9} - 4x + C = \frac{4\tan(9x)}{9} - 4x + C\)

\(\int \frac{4[\tan^2(9x) + 1]}{\csc^2(9x)}\, dx = \dfrac{4\tan(9x)}{9} - 4x + C\)

Topic 8 — Trigonometric Substitution

Use substitutions \(x = a\sin\theta\), \(x = a\tan\theta\), or \(x = a\sec\theta\) to handle \(\sqrt{a^2 - x^2}\), \(\sqrt{a^2 + x^2}\), and \(\sqrt{x^2 - a^2}\) respectively.

Question 8

Use the three trigonometric substitutions (if required) to evaluate \[ \int \frac{-3x}{\sqrt{9 - x^2}}\, dx \] Use \(C\) for the constant of integration. Write the exact answer. Do not round.

Walkthrough

Step 1 — Recognize the form \(\sqrt{9 - x^2}\).

This suggests \(x = 3\sin\theta\), \(dx = 3\cos\theta\, d\theta\), \(\sqrt{9-x^2} = 3\cos\theta\).

Step 2 — Substitute.

\[ \int \frac{-3(3\sin\theta)}{3\cos\theta} \cdot 3\cos\theta\, d\theta = \int -9\sin\theta\, d\theta \]

Step 3 — Integrate.

\(\int -9\sin\theta\, d\theta = 9\cos\theta + C\)

Step 4 — Back-substitute using \(\cos\theta = \frac{\sqrt{9-x^2}}{3}\).

\(9 \cdot \frac{\sqrt{9-x^2}}{3} + C = 3\sqrt{9-x^2} + C\)

\(\int \frac{-3x}{\sqrt{9-x^2}}\, dx = 3\sqrt{9-x^2} + C\)

Topic 9 — Secant Powers via Substitution

For even powers of secant, factor out \(\sec^2 x\) and use \(\sec^2 x = 1 + \tan^2 x\), then substitute \(u = \tan x\).

Question 9

Evaluate the indefinite integral \[ \int 16 \sec^4(8x)\, dx \] Use \(C\) for the constant of integration. Write the exact answer. Do not round.

Walkthrough

Step 1 — Rewrite using \(\sec^2(8x) = 1 + \tan^2(8x)\).

\(\sec^4(8x) = [1 + \tan^2(8x)]\sec^2(8x)\)

Step 2 — Substitute \(u = \tan(8x)\), \(du = 8\sec^2(8x)\, dx\).

\[ \int 16[1 + u^2]\, \frac{du}{8} = 2\int (1 + u^2)\, du = 2\!\left( u + \frac{u^3}{3} \right) + C \]

Step 3 — Back-substitute.

\(2\tan(8x) + \frac{2\tan^3(8x)}{3} + C\)

\(\int 16\sec^4(8x)\, dx = 2\tan(8x) + \dfrac{2\tan^3(8x)}{3} + C\)

Topic 10 — Numerical Integration (Simpson's Rule)

Approximate definite integrals using Simpson's Rule: \(\int_a^b f(x)\, dx \approx \dfrac{\Delta x}{3}[f(x_0) + 4f(x_1) + 2f(x_2) + \cdots + f(x_n)]\) with \(n\) even.

Question 10

Use Simpson's Rule with \(n = 8\) to approximate the integral \[ \int_{0}^{4} 7x\sqrt{x^2 + 1}\, dx \] Round any intermediate calculations, if needed, to no less than six decimal places, and round your final answer to four decimal places.

Walkthrough

Step 1 — Calculate \(\Delta x\).

\(\Delta x = \frac{4 - 0}{8} = 0.5\)

Step 2 — Evaluate \(f(x) = 7x\sqrt{x^2+1}\) at each node.

\(x_0 = 0{:}\ f(0) = 0\)

\(x_1 = 0.5{:}\ f(0.5) = 3.5\sqrt{1.25} \approx 3.905869\)

\(x_2 = 1.0{:}\ f(1) = 7\sqrt{2} \approx 9.899495\)

\(x_3 = 1.5{:}\ f(1.5) = 10.5\sqrt{3.25} \approx 19.022688\)

\(x_4 = 2.0{:}\ f(2) = 14\sqrt{5} \approx 31.304952\)

\(x_5 = 2.5{:}\ f(2.5) = 17.5\sqrt{7.25} \approx 47.431832\)

\(x_6 = 3.0{:}\ f(3) = 21\sqrt{10} \approx 66.433283\)

\(x_7 = 3.5{:}\ f(3.5) = 24.5\sqrt{13.25} \approx 89.321928\)

\(x_8 = 4.0{:}\ f(4) = 28\sqrt{17} \approx 115.524063\)

Step 3 — Apply Simpson's Rule formula.

\[ \int_0^4 \approx \frac{0.5}{3}\bigl[f_0 + 4f_1 + 2f_2 + 4f_3 + 2f_4 + 4f_5 + 2f_6 + 4f_7 + f_8\bigr] \]

\(= \dfrac{1}{6}\bigl[0 + 15.623476 + 19.798990 + 76.090752 + 62.609904 + 189.727328 + 132.866566 + 357.287712 + 115.524063\bigr]\)

\(= \dfrac{969.528791}{6} \approx 161.6547\)

The approximation is \(\approx 161.6547\)
