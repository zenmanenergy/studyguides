AP Computer Science Principles -- Unit 5: Data and Analysis

Why Data Matters

Computing has transformed how data is collected, stored, and analyzed. Vast amounts of digital data are generated every second -- by sensors, apps, social media, transactions, and scientific instruments.

Data is only useful if it is collected carefully, analyzed correctly, and interpreted honestly. Bad data collection, flawed analysis, and misrepresentation of results can cause real harm -- in medicine, justice, finance, and policy.
  • Metadata: data about data. A photo file contains not just pixels but also the time, location, and device it was taken with. Metadata can reveal patterns even without accessing the primary data.
  • Structured data: organized in rows and columns (spreadsheets, databases)
  • Unstructured data: text, images, audio, video -- harder to analyze automatically

Data Collection and Bias

How data is collected determines what conclusions can be drawn. Poor collection introduces bias that distorts results.

Bias TypeDescriptionExample
Sampling biasThe sample doesn't represent the full populationSurveying only college students about retirement savings
Selection biasOnly certain kinds of data get recordedMedical studies that only enrolled men historically
Confirmation biasCollecting/emphasizing data that confirms existing beliefsStopping a study when it shows the desired result
Measurement biasThe measurement tool itself skews resultsA scale that always reads 2 lbs heavy
Algorithmic biasPatterns in training data cause models to treat groups unequallyFacial recognition that is less accurate for darker skin tones

Visualizing Data

Visualization transforms raw numbers into patterns a human can perceive. Choosing the right chart type matters.

Chart Types and When to Use Them

Chart TypeBest For
Bar chartComparing discrete categories
Line chartShowing trends over time
Scatter plotShowing correlation between two numeric variables
Pie / DoughnutShowing parts of a whole (use sparingly; hard to compare slices)
HistogramDistribution of a single numeric variable
Heat mapTwo-dimensional patterns in large datasets
Misleading visualizations are a common manipulation tactic. Watch for: truncated y-axes that exaggerate differences, inconsistent scales, cherry-picked time ranges, and 3D charts that distort proportions.

Correlation vs. Causation

Two variables that move together are correlated. This does NOT mean one causes the other.

  • Correlation: Ice cream sales and drowning rates are both higher in summer -- they are correlated.
  • Causation: Hot weather (a third variable) causes both. Ice cream does not cause drowning.
  • Establishing causation requires controlled experiments where all variables except one are held constant
  • Observational data (surveys, logs, social media) can show correlation but rarely proves causation

Privacy and Data Collection

Every digital interaction generates data. This creates powerful tools -- and serious privacy risks.

ConcernDescription
Personally identifiable information (PII)Name, SSN, email, location, biometrics -- data that can identify an individual
Data aggregationCombining individually harmless data points to reveal sensitive information
Third-party data sharingApps and websites selling user data to advertisers and data brokers
Data breachesUnauthorized access to stored personal data; affects millions at a time
Surveillance capitalismBusiness model based on collecting and monetizing behavioral data
Anonymizing data is harder than it seems. Researchers have repeatedly re-identified "anonymous" users by combining datasets. In one study, just 4 location data points were enough to uniquely identify 95% of people in a dataset.

Machine Learning and Pattern Recognition

Machine learning algorithms find patterns in large datasets and use them to make predictions or decisions without being explicitly programmed for each case.

  • Training data: the dataset the model learns from. If this is biased, the model will be biased.
  • Classification: predicting which category an input belongs to (spam/not spam; disease/no disease)
  • Regression: predicting a numeric value (house price, temperature tomorrow)
  • Machine learning models can achieve high accuracy but are often difficult to interpret -- the "black box" problem
  • High-stakes uses (sentencing, hiring, lending) raise ethical concerns when models produce unexplainable decisions
