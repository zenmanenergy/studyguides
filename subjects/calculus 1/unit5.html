Calculus 1 — Unit 5: Analytical Applications of Differentiation

Extreme Value Theorem and Critical Points

Extreme Value Theorem (EVT): If \(f\) is continuous on \([a,b]\), then \(f\) attains both an absolute maximum and minimum on \([a,b]\).
  • Critical point: \(x = c\) where \(f'(c) = 0\) or \(f'(c)\) is undefined
  • Absolute extrema on a closed interval: evaluate \(f\) at all critical points and endpoints; largest = max, smallest = min

Mean Value Theorem

If \(f\) is continuous on \([a,b]\) and differentiable on \((a,b)\), then there exists \(c \in (a,b)\) such that: \[ f'(c) = \frac{f(b) - f(a)}{b - a} \]
  • The instantaneous rate of change equals the average rate of change at some point
  • Geometrically: a tangent line parallel to the secant exists somewhere on the interval
  • Rolle's Theorem (special case): if \(f(a) = f(b)\), then \(f'(c) = 0\) for some \(c\)

First and Second Derivative Tests

TestConditionConclusion
First Derivative Test\(f'\) changes + to − at \(c\)Local maximum at \(c\)
\(f'\) changes − to + at \(c\)Local minimum at \(c\)
Second Derivative Test\(f'(c)=0\) and \(f''(c) > 0\)Local minimum
\(f'(c)=0\) and \(f''(c) < 0\)Local maximum
  • If \(f''(c) = 0\), the second derivative test is inconclusive — use the first derivative test

Concavity and Inflection Points

  • Concave up when \(f'' > 0\) (holds water); concave down when \(f'' < 0\) (spills water)
  • Inflection point: where concavity changes (i.e., \(f''\) changes sign)
  • \(f''(c) = 0\) is necessary but not sufficient for inflection — concavity must actually change

Optimization

  • Write an objective function to maximize or minimize
  • Express in one variable using a constraint equation
  • Find critical points via \(f' = 0\); verify with first or second derivative test
  • Check endpoints if the domain is closed/bounded
Example: maximize area of rectangle with perimeter 40:
\(P = 2l + 2w = 40 \Rightarrow w = 20 - l\)
\(A = l(20-l) = 20l - l^2,\quad A' = 20 - 2l = 0 \Rightarrow l = 10\) (square)
