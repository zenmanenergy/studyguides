Topic 1 — Construct Parametric Equations for a Line

A parametric line with slope \(m\) satisfies \(\dfrac{dy/dt}{dx/dt} = m\). If \(y\) is given as a function of \(t\), find \(dx/dt\) from the slope condition, integrate, and use the given point to find the constant.

Question 1

Construct parametric equations describing the graph of the line with the following attributes. \[ \text{Slope} = 18 \quad \text{and passing through} \quad (18, -7) \] If \(y = 4 + t\), find the parametric equation for \(x\).

Walkthrough

Step 1 — Find \(dx/dt\) from the slope condition.

Since \(y = 4 + t\), \(dy/dt = 1\). Slope \(= \dfrac{dy/dt}{dx/dt} = 18 \Rightarrow dx/dt = \dfrac{1}{18}\).

Step 2 — Integrate to find \(x(t)\).

\(x = \dfrac{t}{18} + C\)

Step 3 — Use the given point to find \(C\).

At \((18, -7)\): solve \(y = 4 + t = -7 \Rightarrow t = -11\). Then \(18 = \dfrac{-11}{18} + C \Rightarrow C = \dfrac{335}{18}\).

Step 4 — Write the result.

\[ x = \frac{t + 335}{18} \]

\(x = \dfrac{t + 335}{18}\)

Topic 2 — Convert Polar to Cartesian Coordinates

Given polar \((r, \theta)\), convert using \(x = r\cos\theta\) and \(y = r\sin\theta\). A negative \(r\) reflects the point through the origin relative to \(\theta\).

Question 2

Convert the point from polar to Cartesian coordinates. Write your answer in the simplest form or rounded to the nearest hundredth. \[ \left(-6,\ -\frac{\pi}{4}\right) \]

Walkthrough

Step 1 — Recall exact values at \(-\pi/4\).

\(\cos\!\left(-\dfrac{\pi}{4}\right) = \dfrac{\sqrt{2}}{2}\), \quad \(\sin\!\left(-\dfrac{\pi}{4}\right) = -\dfrac{\sqrt{2}}{2}\)

Step 2 — Apply conversion formulas with \(r = -6\).

\(x = -6\cdot\dfrac{\sqrt{2}}{2} = -3\sqrt{2} \approx -4.24\)

\(y = -6\cdot\left(-\dfrac{\sqrt{2}}{2}\right) = 3\sqrt{2} \approx 4.24\)

Cartesian coordinates: \(\left(-3\sqrt{2},\ 3\sqrt{2}\right) \approx (-4.24,\ 4.24)\)

Topic 3 — Slope of the Tangent to a Polar Curve

For \(r = f(\theta)\), treat \(x = r\cos\theta\) and \(y = r\sin\theta\) as parametric in \(\theta\): \[ \dfrac{dy}{dx} = \dfrac{r'\sin\theta + r\cos\theta}{r'\cos\theta - r\sin\theta}, \quad r' = \dfrac{dr}{d\theta} \]

Question 3

Find the slope of the line tangent to the polar curve \[ r = 2\sec^2\theta \] at the point \(\theta = \dfrac{3\pi}{4}\). Write the exact answer. Do not round.

Walkthrough

Step 1 — Compute \(r'\).

\(r' = 4\sec^2\theta\tan\theta\)

Step 2 — Evaluate at \(\theta = 3\pi/4\).

\(\sec(3\pi/4) = -\sqrt{2}\), \(\sec^2(3\pi/4) = 2\), \(\tan(3\pi/4) = -1\)

\(r = 2(2) = 4\), \quad \(r' = 4(2)(-1) = -8\)

\(\sin(3\pi/4) = \dfrac{\sqrt{2}}{2}\), \quad \(\cos(3\pi/4) = -\dfrac{\sqrt{2}}{2}\)

Step 3 — Compute numerator and denominator.

Numerator: \((-8)\!\left(\dfrac{\sqrt{2}}{2}\right) + (4)\!\left(-\dfrac{\sqrt{2}}{2}\right) = -4\sqrt{2} - 2\sqrt{2} = -6\sqrt{2}\)

Denominator: \((-8)\!\left(-\dfrac{\sqrt{2}}{2}\right) - (4)\!\left(\dfrac{\sqrt{2}}{2}\right) = 4\sqrt{2} - 2\sqrt{2} = 2\sqrt{2}\)

Step 4 — Divide.

\[ \frac{dy}{dx} = \frac{-6\sqrt{2}}{2\sqrt{2}} = -3 \]

The slope of the tangent at \(\theta = \dfrac{3\pi}{4}\) is \(-3\).

Topic 4 — Arc Length of a Polar Curve

The arc length of \(r = f(\theta)\) from \(\alpha\) to \(\beta\) is \(L = \displaystyle\int_\alpha^\beta \sqrt{r^2 + (r')^2}\, d\theta\). Simplify the radicand using Pythagorean identities before integrating.

Question 4

Find the arc length of the polar curve \[ r = 16\sin\theta, \quad 0 \leq \theta \leq \frac{\pi}{4}. \] Write the exact answer. Do not round.

Walkthrough

Step 1 — Find \(r'\).

\(r' = 16\cos\theta\)

Step 2 — Simplify \(r^2 + (r')^2\).

\[ 256\sin^2\theta + 256\cos^2\theta = 256 \]

Step 3 — Evaluate the integral.

\[ L = \int_0^{\pi/4} 16\, d\theta = 16\cdot\frac{\pi}{4} = 4\pi \]

\(L = 4\pi\)

Topic 5 — Horizontal Tangent Lines to a Parametric Curve

A horizontal tangent exists where \(dy/dt = 0\) and \(dx/dt \ne 0\). If \(dy/dt\) is never zero on the given domain, the answer is None.

Question 5

Find the equations of any horizontal tangent lines to the curve \[ x = 3t^2, \quad y = \frac{1}{t + 6}, \quad t > -6. \] Write the exact answer. Do not round. If there are no horizontal tangent lines, write None.

Walkthrough

Step 1 — Differentiate \(y\).

\[ \frac{dy}{dt} = -\frac{1}{(t+6)^2} \]

Step 2 — Set \(dy/dt = 0\) and solve.

\(-\dfrac{1}{(t+6)^2} = 0\) has no solution; the fraction is never zero.

None

Topic 6 — Find dy/dx for a Parametric Curve at a Given Point

Compute \(\dfrac{dy}{dx} = \dfrac{dy/dt}{dx/dt}\). First determine which \(t\) corresponds to the given Cartesian point, then evaluate the derivatives there.

Question 6

Find the value of \(\dfrac{dy}{dx}\) for the curve \[ x = 4te^{6t}, \quad y = e^{-8t} \] at the point \((0, 1)\). Write the exact answer. Do not round.

Walkthrough

Step 1 — Find \(t\) at \((0, 1)\).

\(y = e^{-8t} = 1 \Rightarrow t = 0\). Check: \(x = 4(0)e^0 = 0\). Confirmed.

Step 2 — Differentiate.

\(\dfrac{dx}{dt} = 4e^{6t}(1 + 6t)\)

\(\dfrac{dy}{dt} = -8e^{-8t}\)

Step 3 — Evaluate at \(t = 0\).

\(dx/dt = 4\), \quad \(dy/dt = -8\)

\[ \frac{dy}{dx} = \frac{-8}{4} = -2 \]

\(\dfrac{dy}{dx} = -2\)

Topic 7 — Convert a Polar Equation to Rectangular Form

Use \(r^2 = x^2 + y^2\). For a constant \(r = k\), square both sides to get \(x^2 + y^2 = k^2\).

Question 7

Rewrite the following polar equation in rectangular form. \[ r = 16 \]

Walkthrough

Step 1 — Square both sides.

\[ r^2 = 256 \]

Step 2 — Substitute \(r^2 = x^2 + y^2\).

\[ x^2 + y^2 = 256 \]

Rectangular form: \(x^2 + y^2 = 256\) (a circle of radius 16 centered at the origin)

Topic 8 — Parametric Modeling of Projectile Motion

Decompose initial velocity: \(v_x = v_0\cos\alpha\), \(v_y = v_0\sin\alpha\). Then \(x(t) = v_x t\) and \(y(t) = y_0 + v_y t - 16t^2\) (using \(g = 32\) ft/s\(^2\)).

Question 8

On her morning paper route Amy throws a newspaper from her car window 5 feet from the ground. The paper has an initial velocity of \(29\text{ ft/s}\) and is tossed at an angle of \(27^\circ\) from the horizontal. Construct a set of parametric equations modeling the path of the newspaper. Round all final values to the nearest tenth.

Walkthrough

Step 1 — Decompose the initial velocity.

\(v_x = 29\cos 27^\circ \approx 29(0.8910) \approx 25.8\) ft/s

\(v_y = 29\sin 27^\circ \approx 29(0.4540) \approx 13.2\) ft/s

Step 2 — Write the parametric equations.

Initial height \(y_0 = 5\) ft.

\[ x(t) = 25.8t \]
\[ y(t) = 5 + 13.2t - 16t^2 \]

\(x(t) = 25.8t\), \quad \(y(t) = 5 + 13.2t - 16t^2\)

Question 9

Construct a polar equation for the conic section with the focus at the origin and the following eccentricity and directrix.

Walkthrough

Step 1 — Identify \(e\) and \(d\).

\(e = 1\) (parabola). Directrix \(x = -6\) is to the left of the focus, so \(d = 6\).

Step 2 — Select the correct form.

Walkthrough
