Calculus 1 — Unit 7: Differential Equations

Separable Differential Equations

  • A differential equation where variables can be separated to each side
  • Form: \(\dfrac{dy}{dx} = f(x)\cdot g(y)\)
Separate: \(\dfrac{dy}{g(y)} = f(x)\,dx\)
Then integrate both sides, solve for \(y\), apply initial condition if given
  • Example: \(\dfrac{dy}{dx} = xy \;\Rightarrow\; \dfrac{dy}{y} = x\,dx \;\Rightarrow\; \ln|y| = \tfrac{x^2}{2} + C \;\Rightarrow\; y = Ae^{x^2/2}\)

Exponential Growth and Decay

\[ \frac{dy}{dt} = ky \;\Longrightarrow\; y = y_0 e^{kt} \]
  • \(k > 0\): exponential growth (population, compound interest)
  • \(k < 0\): exponential decay (radioactive decay, cooling)
  • \(y_0 = y(0)\) is the initial value; \(k\) is the growth/decay constant
  • Half-life: \(t_{1/2} = \dfrac{\ln 2}{|k|}\)
  • Doubling time: \(t_d = \dfrac{\ln 2}{k}\)

Slope Fields

  • A visual representation of a differential equation \(\dfrac{dy}{dx} = F(x,y)\)
  • At each point \((x,y)\), draw a short segment with slope \(F(x,y)\)
  • Solution curves flow along the direction of the slopes
  • Used to sketch general solution behavior without solving analytically
For \(\dfrac{dy}{dx} = x\): slopes increase with \(x\), independent of \(y\) → solutions are parabolas \(y = \tfrac{x^2}{2} + C\)

Euler's Method

  • Numerical approximation of a DE solution using tangent line steps
  • Given \(\dfrac{dy}{dx} = F(x,y)\), step size \(h\), starting point \((x_0, y_0)\):
\[ x_{n+1} = x_n + h \qquad y_{n+1} = y_n + h \cdot F(x_n, y_n) \]
  • Smaller step size \(h\) → more accurate approximation
  • Euler's method follows the tangent line, so it overshoots on concave up curves
